Compact Sample

Open this sample in Visual Studio

The PostSharp.Samples.Compact sample is a trivial demonstration that PostSharp Laos works also with the Compact Framework.

Note that this project references the assemblies PostSharp.Public.CF.dll and PostSharp.Laos.CF.dll

See the Silverlight sample for a more interesting example of using PostSharp with other platforms than the full .NET Framework.
